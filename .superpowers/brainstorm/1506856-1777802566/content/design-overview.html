Eclipse Survivors — v2.0 Design Overview

Comprehensive upgrade across all 6 areas. Game feel + visual polish + clarity.

🎯 1. In-Game HUD Game Feel Clarity
Red vignette pulse when HP <25% — danger bleeds into the world
Boss approaching: full-screen red edge flash + animated "⚠ BOSS IN 10s" countdown badge
Active weapon tile highlighted with gold border — always know what's firing
Rift % replaced with animated progress arc, shows time-to-next-wave visually
Curse stacks: escalating skull icons (☠ ☠ ☠) with red glow that intensifies per stack
HP bar color-reacts: cyan (healthy) → orange (mid) → red+pulsing (critical)

⚡ 2. Upgrade Picker Clarity Polish
Show current level on weapon cards: "Magic Bolt lv.2 → 3" so you always know progression state
Stat cards show green delta: "+22% ATK", "+40 HP" — what you're actually getting
Evolution cards get full-width treatment with dual-gradient border + entrance fanfare animation
Rarity tiers: common (teal) / rare (gold) / epic (purple+gold gradient) affect border intensity and glow
Card hover: slight tilt (2°) + colored shadow burst under card — satisfying to browse

💥 3. Game Feel / Combat Feedback Game Feel Polish
Kill streak counter: ×3→×5→×10 floating text with escalating size, color, and screen flash
Boss spawn cinematic: 0.4s camera zoom-out + black letterbox bars + "NIGHT LICH AWAKENS" text
Level-up burst: cyan particle fountain from player position + brief time-slow (0.2s at 0.4× speed)
Weapon-specific muzzle flash: small burst sprite at fire origin per weapon type
Type-specific death: fast enemies leave afterimage trail, tank shatters into hex fragments, boss leaves shockwave ring
Low HP red vignette on canvas edge — world reacts to player state

🌌 4. Main Menu Polish Clarity
Personal best banner — best time, highest kill count, max level reached — always visible below title
Last run summary card: "Last run: 3:41 · lv.12 · 312 kills" with weapon loadout icons
Solo button visually dominant (larger, primary gradient); LAN as secondary option below
Animated background upgrade: faster nebula drift + occasional shooting-star streaks across screen

🏁 5. Pause / End Screens Polish Clarity
Pause screen: add per-weapon DPS/damage-dealt breakdown so you can see what's working
Victory: gold particle rain + "RITUAL COMPLETE" fanfare text animation
Defeat: cinematic dark fade, skull icon, "THE ECLIPSE CLAIMS YOU" — emotionally distinct from victory
"NEW RECORD" badge if any stat beats personal best (time, kills, level)
Upgrade path trail: shows the sequence of weapons/stats you built this run

💬 6. Tooltips & Onboarding Clarity
Hover tooltips on all HUD stats: pips, rift%, curse stacks, act label — each explains itself
Weapon tile tooltip: weapon description, current stats (damage, fire rate, range), upgrade history
New act banner: brief slide-in ribbon "ACT II — The Convergence" when act changes
First-run only: subtle pulsing indicator on XP bar and weapon tile to draw attention
